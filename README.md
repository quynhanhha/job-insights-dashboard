# Job Insights Dashboard

Analyze Melbourne-based SWE & DA internship listings to uncover top in-demand skills.

---

## Overview
A small data-to-dashboard pipeline that fetches real job postings, extracts common technical skills, and visualizes them interactively.

## Stack
- Python (pandas, requests)
- Streamlit (frontend)
- GitHub Actions (CI/CD)
